# BlankTab

Chrome 插件

# 生成静态资源

```bash
# 安装依赖
npm install 

# 编译生成
$ gulp 

# 监听自动生成 开发过程中使用
$ gulp watch

# 静态资源  拷贝到生成的对应目录中
$ gulp copy

```
